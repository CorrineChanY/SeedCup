比赛相关

获取排行榜

基本信息

Path： /api/rank

Method： GET

接口描述：

请求参数

Query

参数名称 是否必须 示例 备注
gameStatus 是 1 1-初赛，2-复赛，3-决赛

返回数据

名称类型是否必须默认值备注其他信息
codestring非必须
messagestring非必须
dataobject非必须
├─ sheetInfoobject []非必须表示每项有哪些字段

item 类型: object

├─ namestring必须字段含义
├─ indexstring必须字段名
├─ typestring必须字段类型
├─ sheetDataobject []非必须

item 类型: object

├─ teamIdnumber必须队伍id
├─ teamNamestring必须队伍名
├─ finalScorenumber必须队伍最终得分

用户相关

搜索用户

基本信息

Path： /api/user/search

Method： GET

接口描述：

搜索结果不包含管理员用户和当前用户

请求参数

Query

参数名称 是否必须 示例 备注
keyword 是 hust 搜索关键词，搜索范围是用户名、电话号码、邮箱

返回数据

名称类型是否必须默认值备注其他信息
codestring必须正常返回100，未登录时返回104
messagestring必须
dataobject []必须搜索结果，是一个包含user的list

item 类型: object

├─ idnumber非必须
├─ usernamestring非必须
├─ phoneNumberstring非必须
├─ schoolstring非必须
├─ collegestring非必须
├─ classNamestring非必须
├─ emailstring非必须
├─ createdTimestring非必须
├─ teamIdnumber非必须
├─ adminboolean非必须

删除队员

基本信息

Path： /api/team/member/delete/{userId}

Method： POST

接口描述：

必须在队伍内，并且是队长，报错信息同修改队伍信息接口~~~~
其次，该用户id不能是自己，报错信息同添加队员接口
其次，目标用户必须和当前用户在同一队伍内，否则
{
"code":"106",
"message":"permission denied",
"data":null
}

请求参数

Headers

参数名称 参数值 是否必须 示例 备注
Content-Type application/x-www-form-urlencoded 是

路径参数

参数名称 示例 备注
userId 56 整形，待删除用户id

返回数据

名称类型是否必须默认值备注其他信息
codestring必须成功为100
messagestring必须成功为success
datastring非必须均为null

队伍创建

基本信息

Path： /api/team/sign_up

Method： POST

接口描述：

创建队伍默认队长为队伍的创建者，即当前用户

当已有队伍时，报错：
{
"code":"109",
"message":"already in team",
"data":null
}

请求参数

Headers

参数名称 参数值 是否必须 示例 备注
Content-Type application/json 是

Body

名称类型是否必须默认值备注其他信息
teamNamestring必须队伍名称
highestGradeinteger必须队伍最高年级
introductionstring必须队伍介绍

返回数据

名称类型是否必须默认值备注其他信息
codestring必须成功为100，有重复信息时为101
messagestring必须
datastring []非必须成功时为null，有重复信息时为重复字段list

item 类型: string

├─ 非必须
